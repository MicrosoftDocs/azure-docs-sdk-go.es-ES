| NOMBRE | DESCRIPCIÓN |
|------|-------------|
| [sql/sql](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/sql/sql.go) | Cree y administre nuevos servidores SQL y trabaje con bases de datos en ellos. |
| [sql/queries](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/sql/queries.go) | Cree tablas, agregue filas y consulte datos en Azure SQL Database. |
